# [eagle_IO](https://eagle-io.readthedocs.io/en/latest/)
read eagle HDF5 with native, multithreaded python.

## Dependencies

eagle_IO depends on `h5py`, [`schwimmbad`](https://schwimmbad.readthedocs.io/en/latest/index.html), and `numpy`
